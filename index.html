    Outil de calepinage de carottages de trémie

    Outil numérique interne — études et ingénierie de chantier

    Logiciel web de planification et d'optimisation du carottage de trémies dans des voiles et dalles en béton armé. Il permet de définir les surfaces à caroter, de générer automatiquement le calepinage des carottages selon les contraintes de ferraillage, d'estimer les rendements de chantier et d'exporter directement les données vers SolidWorks pour la modélisation 3D.

    Modules

    • 🗺️Éditeur 2D — Calepinage des carottages en vue de dessus, gestion des zones d'exclusion et sous-zones
    • 📦Éditeur 3D — Visualisation volumique des couches et carottages dans la dalle béton
    • ♻️Production de déchets — Estimation des masses de carottes et graphiques journalier / cumulatif par couche
    • 📊Synthèse projet — Vue consolidée des temps, rendements, quantités, planning, coûts et déchets par couche
    • 📅Planning — Diagramme de Gantt avec jours fériés français et antécédents paramétrables
    • 💶Coûts — Estimation des coûts MO, coût de revient et prix de vente (TU / TA)
    • 📈Tableaux de rendement — Paramétrage des rendements de carottage par diamètre et contexte (hors Z4 / Z4, maillage)
    • 📤Export SolidWorks — Génération de macro VBA .swb pour modélisation 3D automatique
    • 📐Export AutoCAD — Génération de script .scr pour tracé 2D automatique

    À venir

    Exporter SolidWorks (.swb)

    Génère une macro VBA .swb exécutable dans SolidWorks pour créer automatiquement les carottages en 3D.

    📖 Comment utiliser
    1. Cliquer sur Exporter SolidWorks et télécharger le fichier .swb
    2. Dans SolidWorks : Outils → Macros → Exécuter
    3. Sélectionner le fichier .swb téléchargé
    4. La macro crée automatiquement les carottages dans la pièce active

    ⚠️ Les plans spéciaux ne sont pas encore supportés par cet export.

    ⏱ —

    Exporter AutoCAD 2D

    Génère un script de commandes AutoCAD pour reproduire une couche en 2D : contour + zones d’exclusion + cercles de carottages.

    📖 Méthode 1 — Coller dans le terminal
    1. Cliquer sur Copier les commandes
    2. Dans AutoCAD, cliquer dans la ligne de commande
    3. Coller (Ctrl+V) — AutoCAD exécute chaque commande automatiquement
    📄 Méthode 2 — Fichier .scr
    1. Cliquer sur Télécharger .scr et enregistrer le fichier
    2. Dans AutoCAD : taper SCRIPT dans la ligne de commande
    3. Sélectionner le fichier .scr téléchargé
